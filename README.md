## 📌 Installation & Usage
1. Clone the repository:
   ```sh
   git clone https://github.com/singhabhishek98/Portfolio.git
   ```
2. Navigate to the project directory:
   ```sh
   cd my_portfolio
   ```
3. Install dependencies:
   ```sh
   npm install
   ```
4. Start the development server:
   ```sh
   npm start
   ```
5. Open in browser:
   ```
http://localhost:3000```
